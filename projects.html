Projects

Active, open projects are the heartbeat of any public / private sector partnership. Projects will be central to Public Private Sector operations. These projects will reflect the space with a combination of open approaches to ecosystems, platforms with healthy commercial layers that deliver value but also keep the lights on with healthy revenue generation.

Here are a list of current projects:

  • Simple API - A simple, open format approach to deploying APIs using Github, Amazon EC2 and 3Scale API platform.
  • Open Data Repository - A simple open data converter and storage repository that runs completely on Github, no backend required.
  • PPS API Cache - An external cache solution for web APIs, allowing a federated presence for data APIs in public and private sector.
  • PPS API Community - An external community extension of public or private sector APIs that augment existing read APIs with a community driven write API layer.
  • Interface Commons - A platform that delivers a simple and transparent mechanism for the copyright free sharing and collaborative design of API specifications, interfaces and data models.
  • Affordable Care Act Platform - A community driven approach to providing open data and API resources to support federal and local efforts around the affordable care act (ACA).

These project reflects the mission of Public Private Sector, and will drive many other smaller implementations that evolve the organizations planning, architecture and resources.

If you have a project you would like to suggest, please let us know and we'll discuss the best path forward. If you'd like to sponsor or donate one of the existing projects, there are also opportunities to get involved.
